Detailed Description

System that resolves firing from one ship at an other.

When a weapon (Item) is ready to fire, it will call the fire method in ShipMaster, which will in turn put that item in a list of weapons that are pending to be resolved. ShipWeapons will then resolve the firing when the fireAtShip() method is called.
